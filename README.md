TASK 1: HANGMAN GAME:
Game Overview
A simple text-based Hangman game built in Python. Guess the hidden word one letter at a time before you run out of 6 incorrect guesses!
📋 Features
5 predefined secret words (easy difficulty)
6 incorrect guesses allowed
Real-time display of progress
Win/lose conditions
Clean console interface
🛠️ Key Python Concepts Used
random.choice() - Select random word
while loops - Game loop + guess tracking
if-else statements - Input validation + win/lose logic
Strings - Word manipulation, slicing
Lists - Word bank, guessed letters
String methods: .lower(), .count(), .replace()
Console I/O: input(), print()
🚀 How to Play
Run python hangman.py
Guess letters one at a time (a-z)
Current word, remaining guesses, and used letters are displayed
Win by guessing the full word or lose after 6 wrong guesses

TASK 2:STOCK PORTFOLIO TRACKER:
Overview
A simple console-based Stock Portfolio Tracker built in Python. Track your investments by entering stock symbols and quantities, then see your total portfolio value instantly!
📋 Features
Hardcoded real-time stock prices (AAPL, TSLA, GOOGL, MSFT, AMZN)
Add multiple stocks with custom quantities
Real-time portfolio value calculation
Individual stock breakdown
Optional: Save portfolio to CSV file
Clean, formatted output
🛠️ Key Python Concepts Used
Dictionaries - Stock prices & portfolio storage
input() / print() - User I/O
Arithmetic operations - Value calculations
File handling - CSV export (csv module)
Loops & conditionals - Input validation
String formatting - Clean table display
🚀 How to Use
Run python stock_tracker.py
Enter stock symbol (AAPL, TSLA, etc.) and quantity
Type done when finished adding stocks
View total portfolio value
Choose to save to CSV (optional)

TASK 3:BASIC CHATBOT:
Overview
A simple rule-based chatbot built in Python. Responds to basic greetings and farewells using predefined rules. Perfect for learning fundamental Python concepts!
📋 Features
Handles common phrases: "hello", "how are you", "bye"
Case-insensitive input matching
Continuous conversation loop
Multiple phrase variations supported
Clean exit conditions
🛠️ Key Python Concepts Used
if-elif-else - Rule-based decision making
Functions - Modular code organization
while loops - Continuous conversation
input() / print() - User I/O
String methods: .lower(), .strip(), in operator
🚀 How to Run
Save code as chatbot.py
Run python chatbot.py
Start chatting!
