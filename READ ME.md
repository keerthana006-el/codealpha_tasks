Game Overview
A simple text-based Hangman game built in Python. Guess the hidden word one letter at a time before you run out of 6 incorrect guesses!

📋 Features
5 predefined secret words (easy difficulty)
6 incorrect guesses allowed
Real-time display of progress
Win/lose conditions
Clean console interface
🛠️ Key Python Concepts Used
random.choice() - Select random word
while loops - Game loop + guess tracking
if-else statements - Input validation + win/lose logic
Strings - Word manipulation, slicing
Lists - Word bank, guessed letters
String methods: .lower(), .count(), .replace()
Console I/O: input(), print()
🚀 How to Play
Run python hangman.py
Guess letters one at a time (a-z)
Current word, remaining guesses, and used letters are displayed
Win by guessing the full word or lose after 6 wrong guesses
